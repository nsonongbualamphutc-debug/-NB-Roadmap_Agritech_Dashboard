## 🚀 ขั้นตอนการติดตั้ง

### 1. สร้าง Google Sheet + Apps Script

1. สร้าง Google Spreadsheet ใหม่
2. เมนู **ส่วนขยาย** → **Apps Script** → วาง `Code.gs`
3. เลือกฟังก์ชัน `setupSheets` → กด **▶ Run** → อนุญาตสิทธิ์
4. ระบบสร้าง 3 ชีต: `KPIData`, `KPIHistory`, `Users`

### 2. Deploy เป็น Web App

1. กด **Deploy** → **New deployment** → เว็บแอป
2. เรียกใช้ในฐานะ: **ฉัน** • ผู้เข้าถึง: **ทุกคน**
3. คัดลอก URL ที่ได้

### 3. อัปโหลดขึ้น GitHub Pages

1. สร้าง Repository ใหม่ (Public)
2. อัปโหลด `index.html` และ `roadmap_entry.html`
3. **Settings** → **Pages** → Branch: main → Save
4. ได้ URL: `https://username.github.io/repo-name/`

> **หมายเหตุ:** ไฟล์ที่ดาวน์โหลดจากระบบนี้ใส่ URL ของ Apps Script ให้แล้ว ไม่ต้องแก้ไขเพิ่มเติม

## 📐 โครงสร้างข้อมูล — 4 ยุทธศาสตร์ 11 KPI

### ยุทธศาสตร์ที่ 1 — ส่งเสริมและยกระดับการผลิตสู่มาตรฐานเกษตรมูลค่าสูง
- หน่วยงาน: สำนักงานเกษตรและสหกรณ์จังหวัดหนองบัวลำภู (1 ต.ค. 67 – 30 ก.ย. 69)
- KPI 1.1: มูลค่าสินค้าเกษตรที่ได้รับมาตรฐาน (PGS, GAP, Organic, GI) → เพิ่มขึ้น ≥5%/ปี
- KPI 1.2: จำนวนกลุ่มเกษตรกรที่ได้รับการรับรองมาตรฐาน → เพิ่มขึ้น ≥5%/ปี

### ยุทธศาสตร์ที่ 2 — การแปรรูปและสร้างมูลค่าเพิ่ม
- หน่วยงาน: สำนักงานพัฒนาชุมชนจังหวัดหนองบัวลำภู (1 ต.ค. 67 – 30 ก.ย. 69)
- KPI 2.1: มูลค่าผลิตภัณฑ์แปรรูป → เพิ่มขึ้น ≥5%/ปี
- KPI 2.2: จำนวนผู้ประกอบการที่ได้รับการถ่ายทอดเทคโนโลยีแปรรูป → เพิ่มขึ้น ≥5%/ปี
- KPI 2.3: จำนวนผลิตภัณฑ์แปรรูปที่ได้รับมาตรฐานออกสู่ตลาด → เพิ่มขึ้น ≥5%/ปี

### ยุทธศาสตร์ที่ 3 — การพัฒนาการตลาดและสร้างแบรนด์
- หน่วยงาน: สำนักงานพาณิชย์จังหวัดหนองบัวลำภู (1 ต.ค. 67 – 30 ก.ย. 68)
- KPI 3.1: มูลค่าการจำหน่าย → เพิ่มขึ้น ≥10%/ปี
- KPI 3.2: จำนวนผลิตภัณฑ์ที่ยื่นขอใช้ตรา Nong Bua Lamphu Brand → ≥15%/ปี
- KPI 3.3: ความพึงพอใจผู้บริโภคต่อ Nong Bua Lamphu Brand → ≥90%

### ยุทธศาสตร์ที่ 4 — การสร้างความเข้มแข็งให้เกษตรกรและสถาบันเกษตร
- หน่วยงาน: สำนักงานเกษตรจังหวัดหนองบัวลำภู (1 ต.ค. 67 – 30 ก.ย. 68)
- KPI 4.1: จำนวนเกษตรกรต้นแบบ → เพิ่มขึ้น ≥5%/ปี
- KPI 4.2: สัดส่วนวิสาหกิจชุมชนศักยภาพระดับดี → ≥35.6% ของฐานเดิม
- KPI 4.3: สัดส่วนสหกรณ์เกษตรเข้มแข็งระดับ 1-2 → ≥10%

## ❓ คำถามที่พบบ่อย

**Q: แดชบอร์ดต้อง Login ไหม?**
> ไม่ต้อง เปิดได้ทันที เหมาะฉายนำเสนอหรือส่งลิงก์ให้ผู้บริหาร

**Q: หน่วยงานจะเห็น KPI ของหน่วยงานอื่นไหม?**
> ไม่เห็น ระบบแสดงเฉพาะ KPI ที่รับผิดชอบ ยกเว้น Admin เห็นทั้งหมด

**Q: กรอกข้อมูลแล้ว แดชบอร์ดอัปเดตเมื่อไหร่?**
> อัตโนมัติทุก 30 วินาที หรือกดปุ่ม 🔄 รีเฟรช

**Q: ลืมรหัสผ่าน?**
> ไปที่ Google Sheet → ชีต Users → แก้ไขได้โดยตรง หรือล้าง localStorage (F12 → Console → `localStorage.removeItem('agri_users')`)

**Q: อัปเดต Apps Script แล้วไม่มีผล?**
> ต้องสร้าง Deployment ใหม่ทุกครั้ง (New deployment)
